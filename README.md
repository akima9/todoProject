# todo list project (Laravel8 연습)

**Laravel Framework 연습 프로젝트**

**· 개발인원 :** 1명

**· 개발환경 :** VSCode, MacOS Valet

**· 프로토콜 :** HTTP, TCP/IP

**· 개발언어 :** PHP, Javascript

**· 웹서버 :** Nginx

**· 데이터베이스 :** MySql

**· 상세내용**

1. 간단한 스케쥴러 웹 애플리케이션입니다.

2. laravel/ui 적용 : 회원가입, 로그인, 로그아웃, 이메일인증 구현

3. fullcalendar 적용 : 비동기 방식으로 할일 정보 노출 구현
